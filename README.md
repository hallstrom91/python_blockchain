# `Python Blockchain - Simple blockchain`

### `Setup`
```bash
# Step 1: Clone this project.
$ git clone https://github.com/hallstrom91/python_blockchain.git

# Step 2: Navigate to the root folder in the command line interface (CLI) and create a virtual environment:
$ python -m venv venv

# Step 3: Activate the virtual environment from the project's root folder:
Linux/Mac - $ source venv/bin/activate
Windows - $ .\venv\Scripts\activate

# Step 4: Install project dependencies:
$ pip install -r requirements.txt

Now U can contribute to this project Mr.O... :)
```
